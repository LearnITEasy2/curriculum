## 3Ô∏è‚É£ Control Structures: Loops and Conditionals

### Conditionals (If Statements)

These are like decision-makers. If one thing is true, do this; otherwise, do that.

Example:

```

if (temperature > 20) {
    console.log("It's warm outside!");
} else {
    console.log("It's chilly, wear a jacket.");
}
```

### Loops

Loops help you repeat something again and again without getting tired. Think of it like pressing a ‚Äúreplay‚Äù button!
	1.	For Loop (Counted Repetition)
	‚Ä¢	Use this when you know how many times you want to repeat something.

```

    console.log("Hello, world! " + i);
}
// Output: Hello, world! 1, Hello, world! 2, ..., Hello, world! 5
```


2.	While Loop (Until It‚Äôs Done)  
‚Ä¢	Use this when you don‚Äôt know exactly how many times you need to repeat.

```

while (balloons > 0) {
    console.log("Pop! " + balloons);
    balloons--;
}
// Output: Pop! 5, Pop! 4, ..., Pop! 1
```
